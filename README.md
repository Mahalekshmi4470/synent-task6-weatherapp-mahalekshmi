🌦️ SkyCast – Weather App

📌 Internship Project
Synent Technologies – Web Development Internship  
Task 6: API Integration Project (Weather App)

🎯 Objective
Build a weather application that fetches real-time weather data using a public API and displays it dynamically in the UI.


🚀 Features

Core Features
- Search weather by city name
- Display real-time weather data
- Show temperature, humidity, wind speed, and "feels like" temperature
- Dynamic weather icons
- Error handling for invalid cities
- Loading spinner while fetching data

Additional Features
- Celsius / Fahrenheit toggle
- Recent search history (stored using localStorage)
- Clickable history to re-fetch weather
- Responsive and modern glassmorphism UI
- Clean and interactive design


Technologies Used
- HTML5
- CSS3 (Glassmorphism UI design)
- JavaScript (Fetch API)
- OpenWeatherMap API
- LocalStorage


API Used
   - OpenWeatherMap API  
   - https://openweathermap.org/



📁 Project Structure

├── index.html
├── style.css
├── script.js
├── images/
│ └── default.jpg
└── README.md


How It Works

1. User enters a city name
2. API request is sent to OpenWeatherMap
3. Weather data is fetched and displayed
4. Data includes temperature, humidity, wind, etc.
5. Recent searches are saved in localStorage

How to Run

1. Clone the repository
2. Open project folder in VS Code
3. Run index.html in browser
4. Enter any city name to get weather details
